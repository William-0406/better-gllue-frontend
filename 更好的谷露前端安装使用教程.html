首次安装

收到压缩包后按 6 步安装。重点：选择解压后的 dist-extension 文件夹。

1

解压压缩包

右键「更好的谷露前端.zip」，选择“全部解压缩”。不要直接在 zip 里加载插件。

2

打开扩展管理

在 Edge 地址栏输入 edge://extensions/ 并回车。

3

打开开发人员模式

打开“开发人员模式”，页面会出现“加载解压缩的扩展”。

4

加载插件文件夹

点击“加载解压缩的扩展”，选择解压后文件夹里的 dist-extension。

5

打开谷露

进入 http://your-gllue-host.example.com/crm 并登录谷露账号。

6

点击插件图标

在浏览器工具栏点击“更好的谷露前端”图标，页面出现新版前端即安装成功。

最容易错的一步：第 4 步要选解压后文件夹里面的 dist-extension 子文件夹，不是外层解压文件夹、更不是 zip 本身。选错会加载失败，或装上后点图标“没反应”。
如果插件图标不在工具栏，可以点浏览器右上角“扩展”图标，把“更好的谷露前端”固定到工具栏，之后用起来更方便。

项目图谱：先把项目录进来

“项目图谱”是你自己手动维护的在招项目清单，只存在你这台浏览器本地，不联网、不和别人共享。

1

添加项目

点右上角“添加项目”，填公司、职位、base 地点、状态（进行中/已结束）、负责顾问、备注。

2

筛选 / 切换

顶部按公司 / 职位 / 地点筛选；右上角“图谱 / 列表”切换看关系或看明细。

3

编辑 / 删除

列表里点“编辑 / 删除”，或在图谱视图直接点项目节点编辑。

“生成 100 条测试数据 / 清空全部”是压测按钮，“清空全部”会删掉你所有项目，平时别误点。

图谱怎么玩

图谱把每个项目和它的公司 / 职位 / base 地点连起来；共享同一个公司或职位的项目会连到同一个节点，一眼看出哪些项目在抢同一批人。

公司 职位 base 地点 项目

圈越大＝挂的项目越多。项目节点平时是小点，鼠标悬停或被高亮时才显示名字。

想做的操作
缩放按住 Ctrl（Mac ⌘）+ 滚轮，或右上角 ＋ / － 按钮。普通滚轮＝翻页（方便滚回上面的筛选栏）。
平移画布在空白处按住拖动。
整理布局直接拖动某个节点摆位（拖动不会误触发编辑 / 筛选）。
按公司/职位/地点筛选单击对应的公司 / 职位 / 地点节点。
编辑项目单击该项目节点，弹出编辑窗。
聚焦看关联双击任意节点，视野自动框住它和它直接相连的一圈。
高亮关联鼠标悬停节点，高亮它和相关的节点、连线。
快速定位图谱上方搜索框输入名称，回车定位第一个结果。
全屏右上角“全屏”铺满屏幕；全屏时普通滚轮就能直接缩放，Esc 退出。
回到全览 / 重排右上角“适应画布 / 重新排列”。
接到新需求或新候选人时，在图谱里找到对应的职位 / 公司节点，悬停或双击，就能快速看出你之前做过哪些重合项目、能复用哪批候选人。

脉脉查重（详解）

在脉脉(maimai.cn)看候选人时，帮你快速判断“这个人是不是已经在谷露库里了”。

1

打开人才详情

在脉脉打开某人的人才详情页，左侧浮出可拖动的小面板，自动读取姓名 / 年龄 / 学历 / 公司 / 职位。

2

点“查重”

确认读取无误后，点面板上的“查重”按钮。

3

看结果

分“已在库 / 疑似 / 同名 / 未命中”。已在库会显示最近备注，方便你判断跟进情况。

前提：当前浏览器要已登录谷露且在公司网络内，否则会提示“连接谷露失败 / 连不上谷露”（这不代表库里没这个人）。脉脉拿不到手机号 / 邮箱，只能靠姓名+公司+职位比对，所以结果是线索、不是定论，“疑似 / 同名”务必回谷露再核一遍。

更新插件

1

解压新包

把新的「更好的谷露前端.zip」解压出来，保留其中的 dist-extension。

2

删除旧版本

打开 edge://extensions/，找到旧版“更好的谷露前端”，点击删除。

3

加载新版

点击“加载解压缩的扩展”，选择新版解压后的 dist-extension 文件夹。

常见问题

?

点插件没反应

先刷新谷露页面，再点击插件图标。仍无反应时，回扩展管理页重新加载新版插件。

锁

数据和权限

插件只读取当前谷露账号有权限看到的数据，不保存账号、密码、Cookie 或候选人数据。

图

项目图谱会被别人看到吗

不会。项目图谱只存在你自己浏览器本地，不上传、不共享；换电脑或换浏览器需要重新录入。

脉

脉脉查重点了没反应

确认：这台浏览器已登录谷露、在公司网络内、当前是脉脉的人才详情页。看到“连接谷露失败”多半是没登录谷露。

全

图谱滚不动 / 回不到筛选

鼠标停在图谱上时普通滚轮用来翻页；要缩放请按住 Ctrl/⌘ 再滚，或点右上角“全屏”看大图。

新增、推荐、备注、跟进等写操作仍然在谷露原页面完成。插件负责看得更清楚，不替你直接写入谷露。
